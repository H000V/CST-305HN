## Prerequisites
To run this program, you must have Python installed on your system along with Jupyter Notebook and the following scientific computing libraries:
* `numpy`
* `scipy`
* `matplotlib`
* `jupyter`

## Installation
1. Clone or download this repository to your local machine.
2. Open a terminal or command prompt.
3. Install the required dependencies using pip:
    pip install numpy scipy matplotlib jupyter

## How to Run
1. Navigate to the directory containing the downloaded repository.
2. Launch Jupyter Notebook from the terminal:
    jupyter notebook
3. Your default web browser will open. Navigate to and click on the project's `.ipynb` file to open the interactive notebook.

## Usage Instructions
This program runs interactively within the Jupyter Notebook environment.

1. Select the code cell containing the Python script.
2. Execute the cell by clicking the **Run** button at the top of the screen, or by pressing **`Shift + Enter`** on your keyboard.
3. The notebook will prompt you to enter five specific network parameters right below the cell.
